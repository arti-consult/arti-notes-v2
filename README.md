## Installation

1. Clone the repository:

   ```
   git clone https://github.com/Hans-Breyholtz/starter_server.git
   ```

2. Navigate to the project directory:

   ```
   cd starter_server
   ```

3. Install the required dependencies:

   ```
   npm install
   ```

4. Run the development:

   ```
   npm start
   ```
